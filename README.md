# coloresVHDL

## User Guide

### Initial state

* Mode 1
* RGB LEDs off

### Mode 1

* LEDs on or off
* Channel duties locked
* Keys:

  * **UP**: LEDs on
  * **DOWN**: LEDs off
  * **CENTER**: mode 2

### Mode 2

* LEDs on
* Channel duties unlocked
* Keys:

  * **LEFT/RIGHT**: channel selection ->R<->G<->B<-
  * **UP/DOWN**: channel duty +/-
  * **CENTER**: mode 1

## Ports

### Inputs

* Reset button
* 5 keys keypad: left, center, right, up, down

### Outputs

* 2 RGB LEDs: same settings
